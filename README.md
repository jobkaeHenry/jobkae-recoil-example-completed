# 리코일을 이용한 장바구니 상태관리 강의입니다!

## 실행방법

In the project directory, you can run:

### `yarn start`

개발모드로 실행합니다.\
[http://localhost:3000](http://localhost:3000)주소로 브라우저에서 오픈합니다

### `yarn build`

`build` 폴더에 프로젝트를 빌드합니다.\
프로덕션 모드로 어플리케이션을 빌드합니다
